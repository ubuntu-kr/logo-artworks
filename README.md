# logo-artworks

우분투한국커뮤니티 로고 그래픽 파일 모음 입니다. 사용시 우분투한국커뮤니티의 로고임을 명시 해 주시면 자유롭게 사용 가능합니다.  
로고는 Canonical Ltd.에서 제작한 우분투 로고를 기반으로 제작 되었으며, 우분투 로고의 저작권은 Canonical Ltd.에 있습니다.
